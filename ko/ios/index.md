# 잇다(itda)

**작업과 일정, 그리고 사람을 이어주는 공정관리 앱.**

"잇다"는 한국어로 "연결하다"라는 뜻입니다.
잇다 앱은 작업과 작업, 사람과 일정, 계획과 실행을 이어줍니다.

잇다는 일정과 작업 순서가 중요한 모든 프로젝트에서 활용할 수 있습니다 — 건설, IT, 이벤트 기획, 여행, 축제, 팀 세미나 등 어디서든.

## 주요 기능

### 간트 차트
- 일 / 주 / 월 단위 줌 지원
- 대분류(공정 단계)별 작업 그룹화
- 진행률 수동 입력으로 실시간 추적
- 막대 탭으로 작업 상세 조회 및 편집

### CPM 다이어그램
- 네트워크 다이어그램에서 노드 자유 배치
- ES/EF/LS/LF 자동 계산
- 임계 경로(Critical Path) 자동 감지 (빨간색 강조)
- 병목 구간을 조기에 발견하는 여유시간 분석
- CPM 분석 리포트 자동 생성

### 내보내기
- 간트 차트 / CPM 다이어그램 PDF 인쇄
- 간트 차트 Excel(.xlsx) 내보내기
- 발표 모드 지원

### 오프라인 우선
- 인터넷 없이도 모든 기능 사용 가능
- 연결되면 자동 동기화
- 로그인 없이 사용(로컬 모드) 가능
- Google / Apple 로그인으로 여러 기기 동기화

### 템플릿
- 기본 제공 예시: 목조 건축, 소프트웨어 개발
- 나만의 커스텀 템플릿을 만들어 재사용
- 템플릿으로 새 프로젝트 시작 후 자유롭게 수정

### AI 프로젝트 생성
- 말 한마디로 프로젝트 초안 생성 — AI가 작업 목록을 짜드립니다
- 기간, 시작일, 주요 작업을 입력하면 AI가 편집 가능한 일정을 완성
- 검토, 수정, 저장 — 일반 프로젝트처럼 자유롭게 다룰 수 있습니다
- 모든 사용자에게 무료 일일 쿼터 제공, Pro 사용자는 훨씬 넉넉한 한도
- 로그인과 인터넷 연결 필요

### 작업 대시보드
- 프로젝트를 가로지르는 통합 뷰: 모든 태스크를 한눈에
- 3개 탭: 지연 / 오늘 / 예정 (오늘 기준 ±7일)
- 탭으로 편집, 길게 눌러 완료 토글
- 여러 프로젝트에서 놓치는 마감일 없이

### 데이터 백업
- 모든 데이터를 JSON 파일로 백업
- 언제든 복원 가능 — 인터넷 없는 원격 현장에 적합
- 클라우드 동기화와 별개로 데이터를 안전하게 보관
